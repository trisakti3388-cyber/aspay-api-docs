ASPAY API Documentation

ASPAY exposes deposit, withdrawal, callback, status check, and balance endpoints for merchant integrations. All requests are sent as JSON POST and authenticated using MD5 signatures based on sorted parameters.

Base Domain https://api.aspayx.com/API
Credentials

Merchant ID and Secret Key generated from back office.

Signature Format MD5(field=value&...&token={secretkey})

Authentication & Signature

ASPAY uses MD5 signatures generated from ascending-sorted parameters with token={secretkey} appended at the end.

Signature = MD5(field_1=value_1&field_2=value_2&field_x=value_x&token={secretkey})
